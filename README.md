#  Amazon Clone

##  Description
This is a front-end Amazon Clone website built using HTML, CSS, and JavaScript. It replicates the basic UI and functionality of Amazon like product listing, homepage layout, and responsive design.

##  Features
- Amazon-like homepage design
- Product listing section
- Responsive layout (mobile + desktop)
- Navigation bar with search
- Banner slider (if added)
- Add to cart (if implemented)

##  Technologies Used
- HTML
- CSS
- JavaScript

##  How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
